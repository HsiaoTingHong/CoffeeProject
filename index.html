Coffee Project 品牌介紹

咖啡，就是這麼重要。喝咖啡是許多人每天早上第一件要做的事，咖啡的香氣和味道，總是能讓人心情愉悅，提神醒腦。咖啡計畫是一家專注於咖啡文化的咖啡店。我們為您帶來最優質的咖啡豆，並通過經驗豐富的咖啡師，以最完美的方式為您製作一杯杯美味的咖啡。

Coffee Project 品牌主理人

作為一家咖啡文化的推廣者，我們相信咖啡不僅僅是一種飲品，更是生活的一種態度。在這裡，您可以和朋友一同品嚐咖啡，聊天交流；也可以獨自一人，靜靜地享受一杯咖啡，感受生活的美好。

Coffee Project 冠軍咖啡師

我們的咖啡師都是專業的，擁有豐富的咖啡製作經驗。他們不僅精通咖啡的烘焙和磨豆技術，還能夠根據不同的咖啡豆和口味要求，精確地掌握萃取時間和比例，為您帶來最完美的一杯咖啡。

服務介紹

現煮咖啡

我們的咖啡豆都是經過精密烘焙的，以確保每一顆咖啡豆的風味和口感都能夠完美地展現在您的杯中。咖啡製作過程採用嚴格的標準和程序，確保每一杯咖啡都能夠保持一致的品質和口感。

咖啡豆

咖啡豆的品質，決定了咖啡的味道和風味。因此，我們的咖啡豆都是經過嚴格挑選的，來自世界各地最好的咖啡種植區。我們的咖啡豆種類繁多，包括哥倫比亞、肯亞、巴西、印尼等。不同的咖啡豆，帶來不同的風味和口感，滿足您的各種口味需求。

咖啡濾掛包

除了咖啡，我們還提供販售自家烘焙的咖啡豆及咖啡濾掛包，讓您在品嚐美味的同時，還能將好咖啡分享給親朋好友，愉悅地度過每一刻時光。

線上預訂

除了到店選購咖啡豆及咖啡濾掛包之外，我們還供了線上預訂的方式，讓您可以預先訂購喜歡的咖啡商品，以免現場向隅。
